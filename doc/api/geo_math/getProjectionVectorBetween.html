getProjectionVectorBetween function

Vector2 getProjectionVectorBetween (
  1. Vector2 v1,
  2. Vector2 v2,
  3. {Vector2 offset}
)

두 벡터v1, v2 사이의 투영 벡터를 계산하여 반환. offset이 주어질 경우 더해서 반환.

Implementation

Vector2 getProjectionVectorBetween(Vector2 v1, Vector2 v2, {Vector2 offset}) =>
    (v2 * (v1.dot(v2) / v2.dot(v2))) + (offset ?? Vector2.zero());
