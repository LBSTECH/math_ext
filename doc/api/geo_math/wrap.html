wrap function

num wrap (
  1. num value,
  2. num minimum,
  3. num maximum
)

value를 minimum이상 maximum이하로 제한하여 반환합니다.

Geometry.clamp와 달리 maximum를 넘어가게 되면 value를 maximum로 나눈 나머지를 minimum과 비교합니다.

Implementation

num wrap(num value, num minimum, num maximum) {
  if (minimum == maximum) return minimum;

  final range = maximum - minimum;

  return ((value - minimum) % range + range) % range + minimum;
}
